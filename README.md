# 次に進めるオプション
1. ✅ 行の並び替え（ドラッグ&ドロップ）
DragAndDrop APIを使って、行をドラッグして順序を変更。
2. ✅ 検索／フィルター機能
名前や数値でフィルター → string.Contains() や範囲検索。
3. ✅ 並び替え（ソート）機能
ヘッダをクリックすると昇順/降順切り替え。
4. ✅ 詳細編集ビュー（インスペクタのような右パネル）
選択された行のプロパティを詳細に編集可能に。
5. ✅ エクスポート／インポート（JSON / CSV）
外部ツールとの連携も可能になります。
6. ✅ Undo / Redo 拡張
Undo.RecordObject をさらに詳細に組み込み、操作のUndo履歴を丁寧に残す。
